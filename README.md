# Frankfurt History

Interactive map for exploring historical locations across Frankfurt am Main. Built on data from the [Frankfurt History](https://play.google.com/store/apps/details?id=de.frankfurt.history) app by dotcombinat / Historisches Museum Frankfurt.

**Live:** [frankfurt-history.pages.dev](https://frankfurt-history.pages.dev)

## Features

- 1925 POIs across 6 themes on an interactive MapLibre map
- Self-hosted PMTiles basemap with warm archival color palette
- German/English with URL-driven language toggle
- Deep-linkable articles with image galleries, before/after sliders, lightbox
- Walking navigation (ORS live routes + pre-cached nearest neighbors)
- User geolocation
- Text-to-speech via Web Speech API
- Override system for fixing upstream data (coordinates, translations, texts)

## Stack

TanStack Start, React 19, MapLibre GL, Tailwind CSS v4, Cloudflare Pages + R2

## Setup

```bash
# Archive data from API
uv run python3 archive.py

# Merge data + overrides, build GeoJSON + content
uv run python3 scripts/merge.py
uv run python3 scripts/geojson.py

# Pre-cache walking routes (optional, needs ORS_API_KEY)
uv run python3 scripts/precache_routes.py

# Extract PMTiles basemap for Frankfurt area
pmtiles extract https://build.protomaps.com/YYYYMMDD.pmtiles app/public/frankfurt.pmtiles \
  --bbox=8.3,49.9,9.0,50.3 --maxzoom=15

# Dev server
cd app && bun install && bun run dev
```

## Structure

```
data/                   ← API data (auto-generated by archive.py)
overrides/              ← hand-edited fixes (coordinates, translations)
content/                ← merged output (data + overrides)
app/                    ← TanStack Start web app
scripts/                ← build pipeline (merge, geojson, routes)
```
